## 4가지 축 설명
1. **E/I 축 (External/Internal)**: 
   - E: 외향적 코딩 - 라이브러리/API 적극 사용
   - I: 내향적 코딩 - 직접 구현 선호

2. **S/N 축 (Systematic/Intuitive)**: 
   - S: 체계적 코딩 - 꼼꼼한 예외처리
   - N: 직관적 코딩 - 핵심 로직 집중

3. **T/F 축 (Time/Flow)**: 
   - T: 시간 최적화 우선
   - F: 가독성/흐름 우선

4. **J/P 축 (Judging/Perceiving)**: 
   - J: 계획적 코딩 - 구조화된 접근
   - P: 유연한 코딩 - 실험적 접근

## 출력 형식
- mbtiCode: 4글자 코드 (예: INTJ)
- nickname: 재미있는 별명
- 각 축별 점수 (0-100)와 설명
- 강점 3개, 개선점 2개
- 호환되는 스타일
- 조언
